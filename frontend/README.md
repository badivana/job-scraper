# Frontend

This is the frontend of the AI Job Finder SaaS application, built with Next.js 15, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) to view the app.
