# Logo

## Purpose

To provide clear guidelines for the correct usage of the AI-Powered Job Finder logo, ensuring brand integrity and recognizability.

## Logo Mark and Logotype

The logo consists of two elements:
1. **Mark**: A stylized monogram "JF" (Job Finder) with an abstract arrow pointing upward, symbolizing career growth.
2. **Logotype**: The words "AI-Powered Job Finder" in a clean sans-serif typeface (Inter SemiBold).

The mark and logotype can be used together (lockup) or separately, depending on context.

## Clear Space

To maintain visual impact, keep the area around the logo free of other elements.

- The clear space is defined as the height of the letter "f" in the logotype (or the height of the mark) on all sides.
- No text, graphics, or edges should encroach this space.

## Minimum Size

- For print: 12 mm width (or 9 mm for the mark alone).
- For screen: 24 px width (or 18 px for the mark alone).
- Below these sizes, the logo may become illegible; consider using the mark alone for very small applications (e.g., favicon).

## Color Variations

### Primary Logo
- **On light backgrounds**: Use the full-color version (match in `assets/color_palette.md`).
- **On dark backgrounds**: Use the white version (reverse).

### Monochrome
- Use when color printing is not available.
- Black on white background, or white on black/dark background.

### Restricted
- Do not recreate the logo with other colors.
- Do not apply gradients, patterns, or filters to the logo.

## Incorrect Usage

Do not:
- Stretch, skew, or distort the logo.
- Rotate the logo.
- Change the proportions of the mark or logotype.
- Place the logo on a busy background that reduces legibility.
- Add effects such as drop shadow, outline, or emboss.
- Combine the logo with other icons or marks.
- Use outdated versions of the logo.

## File Formats

We provide the logo in the following formats:
- **SVG** (preferred for web and scalable use)
- **PNG** (transparent background, multiple sizes: 64x128, 128x256, 256x512 px)
- **PDF** (for print)
- **EPS** (for legacy print workflows)

All files are located in the `assets/logo/` directory of the repository.

## Download

Designers and developers can download the official logo assets from the company’s internal asset repository or the public GitHub releases page.
