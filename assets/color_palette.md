## Usage Guidelines

- **Primary Color**: Use Blue‑600 for primary call‑to‑action buttons and key links.
- **Backgrounds**: Use Slate‑50 for page backgrounds; Slate‑100 for cards and modals.
- **Text**: Default text color is Slate‑900 on light backgrounds; use White on dark or primary backgrounds.
- **Borders**: Use Slate‑200 for subtle dividers; Slate‑300 (not listed) for stronger separation if needed.
- **States**:
  - Hover: Lighten primary color by 10% (or use Blue‑500).
  - Pressed: Darken primary color by 10%.
  - Disabled: Use Slate‑400 for text and Slate‑100 for background.

## Accessibility (WCAG)

All color combinations meet WCAG 2.1 AA contrast ratios for normal text (≥4.5:1) and large text (≥3:1). Key pairs:
- White text on Blue‑600: 5.3:1
- Slate‑900 on White: 12.6:1
- White on Slate‑900: 12.6:1
- Green‑500 on White: 6.4:1
- Red‑500 on White: 4.6:1

Always verify contrast when placing text over images or gradients.
